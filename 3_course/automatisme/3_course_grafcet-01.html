Grafcet

L’ensemble des complexes se note C, un élément de cet ensemble est un nombres z de la forme :
z = a + ib avec (a, b) ∈ R².
i² = −1.
a est un réel et s’appelle la partie réelle de z notée : Re(z).
b est un réel c'est la partie imaginaire de z noté : Im(z).

Tout nombre réel appartient à C (faire b = 0). 2) Si a = 0 on dit que z est un imaginaire pur

Option des étapes

A tout nombre complexe z = a + ib, on peut faire correspondre un point M(a; b) dans un plan orthonormal (O, −→u , −→v )On dit que z est l’affixe de M. On écrit alors M(z).

Tout nombre réel appartient à C (faire b = 0). 2) Si a = 0 on dit que z est un imaginaire pur
